    Projeto Pix na SumUp

    Este projeto foi desenvolvido durante o curso de UX/UI Design pela EBAC e teve como objetivo criar um layout para e-mail e uma landing page informativa sobre a funcionalidade do Pix na SumUp.
    A proposta era comunicar os benefícios do Pix para pequenos comerciantes, um público com baixo letramento digital, destacando a ausência de taxas, a disponibilidade 24/7 e o recebimento instantâneo das vendas.
    Para isso, foram projetados materiais visuais acessíveis e intuitivos, garantindo clareza na mensagem e facilitando a adoção do Pix como uma solução para modernizar os negócios desses empreendedores.

    Projeto - Desrotula

    O Desrotula é um aplicativo desenvolvido para facilitar a leitura e interpretação de rótulos alimentares. A ideia surgiu da necessidade de tornar a escolha de produtos mais acessível para pessoas com restrições alimentares ou preferências específicas. No app, os usuários podem selecionar alergias, dietas e preferências, e a plataforma prioriza alimentos que atendem a esses critérios. O projeto foi pensado para garantir uma experiência intuitiva, com um design acessível e funcionalidades que simplificam o dia a dia dos consumidores.

    Website para panificadora Dolce Aroma

    Muitas vezes, quando quero tomar um café em um lugar diferente, me vejo correndo atrás de informações como localização, imagens dos produtos, horário de funcionamento, preços e cardápio, ou até mesmo tendo que ligar para o estabelecimento para obter essas informações. Com base nisso, decidi criar um website para uma padaria fictícia, incorporando elementos que considero importantes para despertar o interesse do usuário e facilitar sua busca. Para este projeto, apliquei conceitos de UX que desenvolvi no curso da EBAC, como a criação de uma protopersona do estabelecimento, estudo de cores, tipografia, benchmarking e definição do tom de voz. Além disso, utilizei elementos de UI para garantir que o site fosse de fácil navegação e intuitivo.
